  本应用非常重视您的个人隐私与数据安全，特此说明如下：

        1. 相册权限
        应用仅申请获取设备内的相册图片资源，用于展示、预览、多选、删除等功能。所有图片处理均在本地设备完成，不会上传任何图片到服务器。
        
        2. 数据收集
        本应用不会收集、存储、上传用户的个人信息、位置信息、照片内容、使用记录等任何隐私数据。
        
        3. 位置信息
        应用仅读取照片自带的地理位置元数据用于展示，不会主动获取设备实时定位，也不会上传位置信息。
        
        4. 第三方服务
        本应用未集成任何第三方统计、广告、追踪或社交SDK
        
        5. 照片删除
        删除操作会直接调用系统相册接口，删除后无法恢复，请谨慎操作。
        
        6. 政策更新
        如本政策有更新，将在应用内提示，不另行单独通知。
